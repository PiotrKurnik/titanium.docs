## About this guide

This guide covers:

 * Creating vertices
 * Connecting vertices by creating edges
 * Node and edge properties
 * Indexing of vertices
 * Indexing of edges
 * Removing vertices
 * Removing edges
 * Loading data in popular graph formats from files

This guide covers Titanium 1.0.x.
