Titanium

Clojure graph library built on top of Aurelius Titan and the Tinkerpop stack

Get Started! » Read doc guides » Join the Mailing List » Contribute »

Powerful

All the features you come to expect from a modern graph library.

Scalable

Elastic and linear scalability of Aurelius Titan.

Expressive

Clojuric API. Easy to use DSL for graph querying and traversal, powered by Gremlin and Pipes.

Storage Backends

Choose between BerkeleyDB Java Edition, Apache Cassandra or Apache HBase.

ACID or Eventual Consistency

Transaction support, different replication models provided by storage engines.
